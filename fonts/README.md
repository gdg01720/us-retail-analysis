# フォントフォルダ

このフォルダには日本語フォントファイルを配置できます（オプション）。

## 日本語フォント（オプション）

`ipaexg.ttf` - IPA exゴシックフォント

### フォントの入手方法

1. [IPA フォントのダウンロード](https://moji.or.jp/ipafont/ipafontdownload/)
2. IPAexゴシック（ipaexg.ttf）をダウンロード
3. このフォルダに配置

### フォントが無い場合

フォントファイルが無い場合でも、アプリケーションは動作します。
システムにインストールされているフォントが自動的に使用されます。

### サポートされるフォント

- Meiryo（Windows）
- Yu Gothic（Windows）
- Hiragino Sans（macOS）
- TakaoGothic（Linux）
- IPAGothic（Linux）

## 注意事項

- フォントファイルは通常GitHubにプッシュしません（ライセンス確認が必要）
- ローカル環境とStreamlit Cloudで異なるフォントが使用される可能性があります
- 商用利用の場合は、フォントのライセンスを確認してください
