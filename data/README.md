# データフォルダ

このフォルダには財務データファイルを配置してください。

## 必要なファイル

`financial_data_us.xlsx` - 米国小売業の財務データ

## データファイルの配置

1. Excelファイルをこのフォルダに配置
2. ファイル名が `financial_data_us.xlsx` であることを確認
3. アプリケーションを起動

## データフォーマット

### 必須列
- 企業名
- 決算年度
- 売上高
- 営業利益
- 営業利益率

### 推奨列
- 売上原価、販管費、売上総利益率、販管費率
- 総資産、純資産、有利子負債、自己資本比率
- 営業CF、投資CF、フリーCF
- 従業員数、棚卸資産回転率、総資産回転率

## 注意事項

- データファイルのサイズが大きい場合は、GitHubにプッシュしないでください
- `.gitignore` でデータファイルを除外することをお勧めします
- Streamlit Cloudでは、別途アップロードするか外部ストレージを使用してください
